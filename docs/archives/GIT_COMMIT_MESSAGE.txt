## Technical Details

### Design Principles
- **Mobile-First**: Base styles for mobile, enhanced for desktop
- **Thumb-Driven Design**: Touch targets ≥44x44px, bottom positioning
- **Glassmorphism**: Backdrop blur effect for modern appearance
- **Responsive**: Fluid transitions between breakpoints
- **Accessible**: WCAG AA+ compliance, ARIA labels, keyboard navigation

### Breakpoints
- `--bp-md: 768px` - Main mobile/desktop threshold
- Mobile: ≤768px (Bottom nav visible, single column gameplay)
- Tablet: 769px-1024px (Hybrid layout)
- Desktop: ≥1025px (Classic 2-column, top nav)

### Key Features
1. **Bottom Navigation Bar** (Mobile only)
   - 4 main routes with icons
   - Active indicator with animated underline
   - Glassmorphism effect (blur + semi-transparent)
   - Smooth slideUp animation on load

2. **Immersive Game Layout** (Mobile optimized)
   - Image maximized (flex: 1.2, ~70% height)
   - Answers bottom-sheet style (flex: 0.8, scrollable)
   - Floating header with game stats
   - Min touch targets 44x44px throughout

3. **Responsive Adaptations**
   - HardMode: 1-col mobile, 2-col desktop (phylogenetic tree hidden on mobile)
   - EasyMode: Vertically stacked on mobile
   - GameHeader: Fixed overlay on mobile, static on desktop
